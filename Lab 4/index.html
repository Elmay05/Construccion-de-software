¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Java se ejecuta en la JVM permitiendo independencia del sistema operativo, y JS se ejecuta en navegadores y entornos como Node.js Java tiene menos errores en tiempo de ejecucion ya que es un lenguaje de tipado estatico y fuerte, JS tiene tipado flexible y dinamico Java se usa en aplicaciones empresariales, backend y desarrollo movil, JS es mas para desarrollo web y frontend Java tiene controles estrictos de seguridad y gestion de memoria, JS es mas vulnerable ya que se ejecuta en navegador Java es mas escalable qeu JS

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

  1. getFullYear(): Devuelve año con 4 dígitos
  2. getMonth(): Devuelve el numero de mes entre 0(enero) y 11(diciembre)
  3. getDate(): devuelve el dia del mes del 1 al 31
  4. getHours(): devuelve la hora actual formato 24 hrs (de 0 a 23)
  5. getMinutes(): devuelve los minutos actuales

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

  1. push(): Sirve para añadir un elemento en el arreglo al final
  2. unshift(): Sirve para añadir un elemento al inicio del arreglo
  3. pop(): elimina el ultimo elemento del arreglo
  4. slice(): sirve para copiar un arreglo sin afectar al original
  5. length: para conocer el tamaño del arreglo

¿Cómo se declara una variable con alcance local dentro de una función?

Estas se declaran cuando se inicia una funcion, haciendo qeu se borre al finalizar la ejecucion, ejemplo function suma_numeros(a,b){ const total = a+b; } una vez se termine de ejecutar la funcion, la variable total desaparece, estas variables pueden ser const o let

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Esto puede implicar que la variable pueda ser modificada mas adelante desde cualquier parte del codigo, las funciones se vuelven un poco independientes ya que dependen de un estado externo (variable var), puede haber problemas a la hora de dar mantenimiento ya que el error puede estar muy lejos de donde se ve el error
